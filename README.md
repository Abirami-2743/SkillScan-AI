# SkillScan AI 🤖

An AI-powered resume screening tool for HR teams.

## Features
- Bulk resume upload (PDF, DOCX, ZIP)
- AI scoring using Groq Llama3
- Auto email to candidates (shortlist/reject)
- Ranked candidate dashboard
- Scan history tracking
- Email verification for HR accounts

## Tech Stack
- **Backend**: Python, FastAPI, MongoDB
- **AI**: Groq API (Llama3-70B)
- **Frontend**: React, TypeScript, Vite
- **Email**: Gmail SMTP
- **Auth**: JWT + Magic Link Verification
